## 🧠 Pattern Summary

| Pattern | Key Idea |
|----------|----------|
| DFS | Recursive traversal (Preorder, Inorder, Postorder) |
| BFS | Queue-based level traversal |
| BST | Left < Root < Right |
| Trie | Prefix-based search |
| Tree Construction | Build tree from traversal orders |
| Backtracking + Trie | Efficient multi-word search |
| Tree DP | Compute answers from child subtrees |
